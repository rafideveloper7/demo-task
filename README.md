## API Notes
The provided APIs have CORS restrictions when accessed from localhost. To handle this:

- The app first tries to connect to the real API
- If CORS blocks it, it shows demo data instead
- The design and all features still work perfectly

If the backend enables CORS, the app will automatically use real data with no code changes needed.
